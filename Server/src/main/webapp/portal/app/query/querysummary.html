师风师德自查统计情况汇总

6月3日对全体教师进行师风师德自我测评，教师们能够做到真实、客观的评价，反应出了许多问题也同时提出了一些宝贵意见，现将教师们反应的问题与解决方案汇总如下：

1.能做到上课前面对镜子正衣冠，保持微笑，以良好的精神面貌进入课堂自查汇总情况；(10分： 人，9～8分： 人，7～6分 人，5分以下： 人)

2. 在平常的工作中，能做到对于学习困难学生、犯了错误的学生不讽刺、挖苦、歧视自查汇总情况；(10分：人，9～8分：人，7～6分：人，5分以下：人)

3.在平时的教学中，能刻苦钻研业务，不断学习新知识，探索教育教学规律，改进教育教学方法。自查汇总情况；(10分：人，9～8分：人，7～6分：人，5分以下：人)

4.在教学中能做到针对不同班级学生特点有针对性的备课上课，因材施教自查汇总情况；(10分：人，9～8分：人，7～6分：人，5分以下：人)

5. 在课后批改作业时能关注到重点学生并有针对性的进行批注,自查情况汇总。(10分：人，9～8分：人，7～6分：人，5分以下：人)

6. 能注重培养学生良好品行，激发学生创新精神，不以分数作为评价学生的唯一标准自查情况汇总(10分：人，9～8分：人，7～6分：人，5分以下：人)

7. 在与同事相处时，能做到尊重同事、团结协作、相互学习、相互帮助，维护其他教师在学生中的威信自查情况汇总。(10分：人，9～8分：人，7～6分：人，5分以下：人)

8.对特殊孩子进行单独辅导与教育自查情况汇总(10分：人，9～8分：人，7～6分：人，5分以下：人)

9. 与所带班级的班主任关于孩子（学习、纪律等）进行沟通了解自查情况汇总(10分：人，9～8分：人，7～6分：人，5分以下：人)

10. 利用家长接送孩子的时机与家长进行有效沟通自查情况汇总(10分：人，9～8分：人，7～6分：人，5分以下：人)
